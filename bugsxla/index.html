created by Phil Woodward

About

BugsXLA is a Microsoft Excel add-in that facilitates Bayesian analysis of GLMMs and other complex model types by providing an easy to use interface for the OpenBUGS, WinBUGS or JAGS packages.

It requires the following to be already loaded on your PC:

  • Windows OS: current version developed on Windows 10.
  • MS Excel: current version developed on Excel 2016 (Office 365).
  • MS Notepad (a standard component of any recent MS operating system)
  • OpenBUGS, WinBUGS or JAGS

Refer to BugsXLA Google Group discussion for help resolving implementation issues.
