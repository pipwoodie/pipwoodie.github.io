Instructions for installing BugsXLA

If you have a previous version of BugsXLA loaded it must be uninstalled from Excel's add-in list. The new version should automatically uninstall all previous versions, but it is recommended that you attempt to do this manually first by opening the 'Options for BugsXLA' form and clicking on the 'Unload BugsXLA' button and confirming 'OK'. Optionally, you may also delete the BugsXLA directory and all the files associated with this previous version.

Installation of New Version

  1. Download the files from the Download page. It is recommended that you extract these to C:\BugsXLA7
  2. Open MS Excel.
  3. Make Excel recognise the BugsXLA add-in:
    • Excel 2000-2003
      1. Select Tools: Add-Ins.
    • Excel 2007
      1. Select MS Office Button : Excel Options : Add-Ins (listed in left hand column)
      2. At the bottom of the form ensure Manage: Excel Add-Ins is selected, and click "Go ..."
    • Excel 2010-2016 (Office 365)
      1. Select File : Options : Add-Ins (listed in left hand column)
      2. At the bottom of the form ensure Manage: Excel Add-Ins is selected, and click "Go ..."
  4. Browse in the directory to which the zipped files have been extracted (see step 1).
  5. Select the file BugsXLA7.xla and make sure that BugsXLA7 has been checked in the Add-Ins dialog box before clicking 'OK'.
  6. If you are warned that OpenBUGS, WinBUGS, JAGS or Notepad is not in 'specified location', click 'Yes' and alter the directory entries so that these programs can be found.
  7. A toolbar titled 'BugsXLA7' should appear that can be moved to your favoured location on the Excel frame. In Excel 2007-2016 the toolbar will appear on the Add-In tab. It might be necessary to exit and restart Excel in order to see the Add-In tab.
  8. If at anytime you have any problems running the program try a clean reinstall by first removing all the old components:
    1. Open Excel and remove the BugsXLA toolbar:
      • Excel 2000-2003
        1. Tools : Customize
        2. Select 'BugsXLA' from the list of Toolbars
        3. Delete (confirm OK)
      • Excel 2007-2016
        1. Right mouse click on any BugsXLA icon and select 'delete custom toolbar'.
    2. Remove the BugsXLA add-in:
      • Excel 2000-2003
        1. Tools : Add-Ins
        2. Uncheck 'BugsXLA' from the list of Add-Ins
        3. OK
      • Excel 2007
        1. Select MS Office Button : Excel Options : Add-Ins (listed in left hand column)
        2. At the bottom of the form ensure Manage: Excel Add-Ins is selected, and click "Go ..."
        3. Uncheck 'BugsXLA' from the list of Add-Ins
        4. OK
      • Excel 2010-2016
        1. Select File : Options : Add-Ins (listed in left hand column)
        2. At the bottom of the form ensure Manage: Excel Add-Ins is selected, and click "Go ..."
        3. Uncheck 'BugsXLA' from the list of Add-Ins
        4. OK
    3. Exit Excel
    4. Delete all the downloaded BugsXLA files from your hard drive. Sometimes Windows has the annoying habit of making copies of files such as Excel add-ins, and so in extreme cases it might be necessary to undertake a complete search of your hard drive in order to locate and delete these copies.
